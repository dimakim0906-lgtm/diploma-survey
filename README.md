# Дипломный проект: Мобильное приложение для опросов на Flutter

**Мобильное приложение для проведения и прохождения опросов**

---

## 📋 О проекте

Дипломная работа представляет собой кроссплатформенное мобильное приложение для создания, проведения и анализа социологических/маркетинговых опросов.

Приложение позволяет:
- Создавать различные типы опросов
- Проходить опросы в удобном интерфейсе
- Просматривать статистику и результаты в реальном времени
- Работать как онлайн, так и в оффлайн-режиме

---

## ✨ Основные возможности

- **Авторизация** пользователей
- **Создание опросов** (разные типы вопросов: одиночный выбор, множественный, текстовый и т.д.)
- **Прохождение опросов** с удобным UI/UX
- **Статистика** и визуализация результатов
- **Административная панель** (для управления опросами)
- Поддержка **Firebase** (аутентификация, база данных, хранилище)

---

## 🛠 Технологии

- **Flutter** (Dart) — кроссплатформенная разработка
- **Firebase** — аутентификация, Firestore, Storage
- **Provider / Riverpod** — управление состоянием
- **MVVM** архитектура
- **Google Fonts, Material 3** — дизайн
